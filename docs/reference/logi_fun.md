# Function for logistic model

Base function for the Logistic model. This function is used in the
Logistic model simulation function
[`sim_logistic()`](https://alvesks.github.io/epifitter/reference/sim_logistic.md)

## Usage

``` r
logi_fun(t, y, par)
```

## Arguments

- t:

  Vector of time

- y:

  Vector of disease intensity

- par:

  List of parameters
