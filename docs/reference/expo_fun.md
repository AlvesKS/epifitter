# Function for Exponential model

Base function for the Exponential model. This function is used in the
Exponential model simulation function
[`sim_exponential()`](https://alvesks.github.io/epifitter/reference/sim_exponential.md)

## Usage

``` r
expo_fun(t, y, par)
```

## Arguments

- t:

  Vector of time

- y:

  Vector of disease intensity

- par:

  List of parameters
