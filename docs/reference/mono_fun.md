# Function for Monomolecular model

Base function for the Monomolecular model. This function is used in the
Monomolecular model simulation function
[`sim_monomolecular()`](https://alvesks.github.io/epifitter/reference/sim_monomolecular.md)

## Usage

``` r
mono_fun(t, y, par)
```

## Arguments

- t:

  Vector of time

- y:

  Vector of disease intensity

- par:

  List of parameters
