# Function for Gompertz model

Base function for the Gompertz model. This function is used in the
Gompertz model simulation function
[`sim_gompertz()`](https://alvesks.github.io/epifitter/reference/sim_gompertz.md)

## Usage

``` r
gompi_fun(t, y, par)
```

## Arguments

- t:

  Vector of time

- y:

  Vector of disease intensity

- par:

  List of parameters
